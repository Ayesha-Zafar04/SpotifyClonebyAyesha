# Spotify Clone - Project Context

## Project Overview

This project is a full-stack Spotify-inspired music streaming web application built for learning modern full-stack development. The goal is to follow professional software engineering practices while creating a scalable and maintainable application.

The project will be developed sprint by sprint using AI-assisted development in Cursor.

---

# Project Goal

Build a responsive Spotify Clone that demonstrates production-level architecture, reusable components, clean code, and modern development practices.

The application should be portfolio-ready and easy to extend in the future.

---

# Development Rules

* Always use TypeScript.
* Follow App Router conventions.
* Keep components reusable.
* Avoid duplicate code.
* Use meaningful file and variable names.
* Keep functions small and focused.
* Write clean, readable code.
* Explain architectural decisions before implementing major changes.
* Do not modify unrelated files.
